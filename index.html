Introduction

Coding, sometimes called computer programming, is how we communicate with computers. Code tells a computer what actions to take, and writing code is like creating a set of instructions. By learning to write code, you can tell computers what to do or how to behave in a much faster way. You can use this skill to make websites and apps, process data, and do lots of other cool things.

It can be really fun and interesting to learn how to code. Being able to code is a great skill to have and it is very useful, especially because coding is used in all electronic devices we use every day! Some examples are mobile phones, laptops, washing machines, television, the ticket machine at the train station and the vacuum cleaner.

Programs We Offer

Scratch Computer Coding

Scratch is a high-level block-based visual programming language aimed primarily at children as an educational tool for programming, with a target audience of ages 5 to 16. Learners can create game, animation and story projects on their computer using a block-like interface. Projects can be exported or converted to standalone HTML5, Android apps, Bundle (macOS) and EXE files using external tools.

MIT App Inventor Coding

MIT App Inventor is an intuitive, visual programming environment that allows everyone even children to build fully functional apps for smartphones and tablets. It is a blocks-based programming which facilitates the creation of complex, high-impact apps in significantly less time. The MIT App Inventor seeks to empower all people, especially young people, to move from technology consumption to technology creation.

Arduino Robotics and Electronics

Arduino is a software as well as hardware platform that helps in making electronic projects. It is an open source platform and has a variety of controllers and microprocessors. There are various projects created with the help of the Arduino. Some of the projects are: Home Automation System using IoT (Internet of Things), smart trash can, fire Detector, automatic street light, car robot etc.

Website Design with HTML/CSS

Website design is the creation of webpages. HTML is the standard markup language for creating Web pages, it is the language of the Web. CSS is the language that is used to style an HTML document. CSS describes how HTML elements should be displayed. It can control the layout of multiple web pages all at once.
